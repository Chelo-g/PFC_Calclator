# PFC Calclator
* 減量時の推奨カロリー、三大栄養素(PFC)の推奨摂取量を計算できます
* 三大栄養素の範囲を指定し、適した食品を検索できます
* 検索結果をピックアップし、合計の栄養素を計算できます
